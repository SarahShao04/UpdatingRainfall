# Replication and Extension of Brückner & Ciccone (2011)
## "Rain and the Democratic Window of Opportunity" — *Econometrica*, 79(3): 923–947

This repository replicates and extends Brückner & Ciccone (2011), which studies how
rainfall-driven income shocks affect democratic change across 41 Sub-Saharan African
(SSA) countries. The original dataset covers 1981–2004. We extend all three core
variables through the most recent year available and re-run the paper's main
regressions on the longer panel.

**Original paper:** [Brückner & Ciccone (2011), *Econometrica*](https://www.econometricsociety.org/publications/econometrica/browse/2011/05/01/rain-and-democratic-window-opportunity)

## Notebooks

### `variable_construction.ipynb`

Constructs and extends the three core variables from B&C (2011):

| Part | Content | Output |
|---|---|---|
| Part 1 | Democracy: Polity V (2005–2018) + V-Dem rescaled (2019–2024) | `democracy_extended_2005_2024.csv` |
| Part 2 | Real GDP per capita: Penn World Table 11.0 (2005–2023) | `pwt_extension_final.csv` |
| Part 3 | Rainfall: NASA GPCP v2.3 country-year aggregation (2005–2024) | `gpcp_extended_2005_2024.csv` |
| Part 4 | Panel integration: merge original B&C data with all extensions | `panel_final.csv` |

### `replication_analysis.ipynb`

Runs the paper's regression tables on both the original 1981–2004 sample and the
extended 1981–2024 sample. Each table has Panel A (original) and Panel B (extended).
A robustness check section compares results across three samples (1981–2004 /
1981–2018 / 1981–2024) to isolate the source of any changes.

| Table | Content |
|---|---|
| Table III | Rainfall and Polity change (OLS) |
| Table IV | Rainfall, per capita GDP, and country-specific recessions (OLS) |
| Table V | Income shocks and Polity change — Log GDP instrument (2SLS) |
| Table VI | Income shocks and Polity change — Recession indicator (2SLS) |
| Table VII | Democratic convergence: lagged Polity2 level as control (OLS + 2SLS) |
| Table VIII | Rainfall and Polity transitions — democratic/autocratic/coup (OLS) |
| Table IX | Income shocks and transitions to democracy (2SLS) |
| Robustness | Three-sample comparison to isolate V-Dem splice noise from structural change |

**Estimation approach:** All specifications include country fixed effects,
country-specific linear time trends, and year fixed effects. Standard errors are
Huber robust, clustered at the country level. For 2SLS, Anderson-Rubin *p*-values
(robust to weak instruments) are reported in square brackets.

## Critical Variable Mapping

The original database stores `polity_change` as a *backward* difference
(D_t − D_{t−1}), while B&C's notation uses a *forward* difference. This creates a
two-period offset between database column names and the paper's labels:

| Paper label | `panel_final.csv` column |
|---|---|
| Log Rainfall, *t* (contemporaneous) | `lgpcp_l` |
| Log Rainfall, *t*−1 (lagged; **2SLS instrument**) | `lgpcp_l2` |
| Log GDP, *t*−1 (Table V endogenous) | `lgdp_l2` |
| Recession indicator, *t*−1 (Table VI endogenous) | `recession_l2` |
| Polity2, *t* (convergence control) | `polity2l` |

## Extension Results: Key Findings

### What holds up

- **Rainfall → democratic transitions** (Table VIII) *strengthens* over the longer
  panel: Log rain, *t*−1 → DemTrans: −0.142\*\*\* vs. −0.125\*\* in original.
- **Income shocks → democratic transitions** (Table IX, 2SLS) also strengthens:
  Log GDP → DemTrans: −2.280\*\*\* [AR *p* = 0.005] vs. −1.285\*\* original.

### What weakens

- **Rainfall → ΔPolity2** (Table III OLS): coefficient shrinks ~60% and loses
  significance (−0.588 vs. −1.461\*\*) over the extended sample.
- **2SLS income shocks → ΔPolity2** (Tables V–VI): no longer significant,
  because the rainfall → GDP first-stage weakens after 2004.

### Robustness check

The 1981–2018 sample (Polity data only, no V-Dem) already shows the same
attenuation, confirming the cause is a **structural weakening of the
rainfall–income transmission channel** after 2004, not V-Dem data splicing.

## How to Reproduce

### Prerequisites

```bash
pip install pandas numpy scipy statsmodels linearmodels xarray netCDF4 \
            geopandas shapely fiona pyproj rtree matplotlib openpyxl
```

### Step 1 — Variable Construction

Run `variable_construction.ipynb` top-to-bottom. Downloads GPCP files from NOAA
NCEI (internet required). Outputs written to `data/processed/`.

### Step 2 — Replication Analysis

Run `replication_analysis.ipynb` top-to-bottom. Requires only
`data/processed/panel_final.csv` and `data/raw/original/database.dta`.
