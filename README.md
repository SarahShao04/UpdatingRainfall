# Updating Rainfall and Climate Data for Sub-Saharan Africa (2005–2018)

## Overview

This project extends the dataset used in Brückner & Ciccone (2011), *"Rain and Democratic Window of Opportunity"* (*Econometrica*, 79(3): 923–947), which studies the effect of rainfall-driven income shocks on democratic change across 41 Sub-Saharan African (SSA) countries from 1981 to 2004.

The original dataset ends in 2004 because that is the last year covered by the data sources available at the time of publication (Polity IV, Penn World Table 6.2, and GPCP through 2004). This repository constructs updated versions of the three core variables — **democracy (Polity)**, **real GDP per capita (Penn World Table)**, and **rainfall (NASA GPCP)** — and extends them through 2018 using the same methodologies described in the original paper.

**Original paper:** [Brückner & Ciccone (2011), Econometrica](https://www.econometricsociety.org/publications/econometrica/browse/2011/05/01/rain-and-democratic-window-opportunity)

## Variables Constructed

### 1. Democracy — Polity2 Extension
- **Source variable:** `polity2` (combined Polity score, ranging from −10 to +10)
- **Also extended:** `exconst` (executive constraints), `polcomp` (political competition), `exrec` (executive recruitment)
- **Method:** The original dataset uses Polity IV scores through 2004. We load the Polity V dataset (`p5v2018.xlsx`), filter to the 41 SSA countries in the B&C sample, and extract 2005–2018 observations. Variable definitions are consistent across Polity IV and V.

### 2. Real GDP per Capita — PWT Extension
- **Source variable:** `rgdpe` (expenditure-side real GDP, millions of 2017 USD) and `pop` (population)
- **Constructed variable:** `rgdpe_pc = (rgdpe × 10⁶) / pop` (real GDP per capita, constant 2017 USD)
- **Method:** The original dataset uses PWT 6.2 (2005 USD base). PWT 11.0 uses a 2017 base; within-version comparisons are valid. We keep all countries from PWT 11.0 for years ≥ 2005.
- **Note:** Cross-version comparisons (PWT 6.2 vs. 11.0) should be treated with caution due to methodology updates.

### 3. Rainfall — GPCP Country-Year Aggregation
- **Source:** NASA GPCP Monthly Precipitation CDR v2.3 (NetCDF, 2.5° × 2.5° grid, units: mm/day)
- **Method:**
  1. Download all 12 monthly `.nc` files for a given year from NOAA NCEI.
  2. Multiply each monthly grid by the number of days in that month to get mm/month.
  3. Sum across 12 months to get annual total rainfall (mm/year) per grid cell.
  4. Spatially join grid centroids to country polygons (Natural Earth 1:50m shapefile).
  5. Average grid cells within each country (simple mean and area-weighted using cos(latitude)).
  6. Filter to the 41 SSA countries in the B&C sample using ISO3 country codes.
- **Output:** `country_rainfall_2005_ssa.csv` (simple mean) and `country_rainfall_weighted_2005_ssa.csv` (area-weighted).

## How to Reproduce

### Prerequisites

This project uses **Python 3.12** (Anaconda distribution recommended).

Install required packages:
```bash
pip install pandas numpy xarray netCDF4 geopandas shapely fiona pyproj rtree matplotlib openpyxl
```

Or with conda:
```bash
conda install pandas numpy xarray netcdf4 geopandas matplotlib openpyxl
pip install rtree
```

**Stata** is required only to open `data/raw/original/database.dta` (or use `pandas.read_stata()`).

### Running the Notebook

1. Clone this repository:
   ```bash
   git clone https://github.com/SarahShao04/UpdatingRainfall.git
   cd UpdatingRainfall
   ```

2. Open the notebook:
   ```bash
   jupyter notebook UpdatingRainfall.ipynb
   ```

3. Run all cells from top to bottom (**Kernel → Restart & Run All**).
   - Cells download GPCP `.nc` files automatically from NOAA NCEI (internet required).
   - All other raw data files are included in `data/raw/`.

4. Outputs are saved to `data/processed/`.

### Note on GPCP Downloads
Cell 9 downloads the 12 monthly GPCP files for 2005 from NOAA NCEI. These are already included in `data/raw/gpcp/2005/`. To extend to additional years, replicate that cell changing the year and base URL accordingly.
